
Emacs client for the wonderful note-tagging system : http://faradaysbook.com/

## INSTALL
    (require 'frb)
    (setq frb-username <your-email-address>)
    (setq frb-password <your-password>)

## COMMANDS

### Editing

     frb-note-buffer
     frb-note-region       - default keybinding is C-c C-f 
     frb-note <message>

### Viewing
     frb-tags
     frb-notes

### Openbook
     frb-open-tags
     frb-open-notes
