
Client to the wonderful note-tagging system : http://faradaysbook.com/

## INSTALL
    (require 'frb)
    (setq frb-username user)
    (setq frb-password user)

## COMMANDS

### Editing

     frb-note-buffer
     frb-note-region 
     frb-note <message>

### Viewing
     frb-tags
     frb-notes

### Openbook
     frb-open-tags
     frb-open-notes
